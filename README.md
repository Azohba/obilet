# Obilet Automation Project


- Web Test Automation Scenario with Java , Gauge, Selenium, Log4j on Maven framework

Automated Scenerio: 

 - Register
 - Buy Plane Ticket

- Jdk 1.8
- Apache Maven 3.6.3
- Gauge v1.4.1

## Setup And Run

- You can run by specs file.
